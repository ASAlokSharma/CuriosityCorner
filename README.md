# CuriosityCorner
A website dedicated to answer the questions that arises when we see and observe the beauty of the universe. To answer that 'WHY' factor.

![Curiosity Corner Logo](WebsiteLogo.png)

**Complex Things, Simply Explained**

Curiosity Corner is an educational website that breaks down complex topics in science, technology, nature, and space into simple, easy-to-understand explanations. Perfect for curious minds of all ages!

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional software or dependencies required!

### Installation

1. **Clone or Download** the repository:
   ```bash
   git clone https://github.com/yourusername/curiosity-corner.git
   ```

2. **Navigate** to the project folder:
   ```bash
   cd curiosity-corner
   ```

3. **Open** `index.html` in your web browser:
   - Double-click the file, or
   - Right-click and select "Open with" your preferred browser

That's it! The website is now running locally.
